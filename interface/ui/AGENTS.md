# UI COMPONENTS

**NOTE:** This directory exists for historical reasons. The actual Qt UI code is located in `../qt/`.

The Qt UI code includes:
- `qt/app.py` — Application entry point
- `qt/window_controller.py` — Main window composition
- `qt/theme.py` — Material Design 3 inspired theme system
- `qt/dialogs/` — Dialog implementations
- `qt/widgets/` — Custom widget implementations

See `../AGENTS.md` for the main module documentation.
